This folder contains the I-BLEND analysis code.
Consider to the privacy, the raw dataset would not be contained in this file
However, you can follow the link to download them.
And these are codes for analyzing boy hostel and girl hostel in I-BLEND.
## File Description
- `config.py`: Defines all simulation parameters and dataset settings.
- `data_load.py`: Loads, cleans, and resamples the I-BLEND electricity load data.
- `main.py`: Runs the full I-BLEND analysis workflow and prints the main results.
- `monte_carlo_simulation_GAP.py`: Implements the Monte Carlo rebound simulation with efficiency gap, rebound rate, aggregation, and pricing response.
- `plot_price_elasticity.py`: Creates the price elasticity response curve used in the methodology section.
- `plots_for_pricing.py`: Creates the dynamic pricing working principle figure.
- `support_plot.py`: Creates additional supporting plots for system impact and pricing mitigation analysis.
